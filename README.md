# Personal-data-protection
Personal data protection

Вам нужно защитить данные клиентов страховой компании Noname. Разработайте такой метод преобразования данных, чтобы по ним было сложно восстановить персональную информацию. Обоснуйте корректность его работы.

Нужно защитить данные, чтобы при преобразовании качество моделей машинного обучения не ухудшилось.
